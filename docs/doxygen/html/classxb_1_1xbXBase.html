◆ GetDefaultIxTagMode()

xbInt16 xb::xbXBase::GetDefaultIxTagMode ( ) const

Get the default index tag mode.

XB_IX_DBASE_MODE - DBASE Compatibility mode
1) For index tags defined as unique, dBASE allows mutliple records in a table all having the same key, but only the first entry in the table with the unique key will be referenced in the unique index.
2) Deleted DBF records are not deleted from the associated tag file.
3) This is the system default.

XB_IX_XBASE_MODE - XBase mode
1) For index tags defined as unique, the library will throw an error when an attempt is made to add/update a record with a duplicate key.
2) Deleted DBF records are deleted from the associated index tag.

Note
When switching between modes, reindex any affected index tags.

◆ GetEndianType()

xbInt16 xb::xbXBase::GetEndianType ( ) const

Get the Endian type.

Endianness is the byte order used by the computer architecture. Big endian stores the most signficant byte first whereas little endian stores the least signficant byte first. The DBF file formats utilize the little endian format and the Xbase64 library automatically handles the endianness of the archticture the software is running on to store the data in little endian format.

Returns
B - Big endian
L - Little endian

◆ SetDefaultIxTagMode()

xbInt16 xb::xbXBase::SetDefaultIxTagMode ( xbInt16  iIndexMode = XB_IX_DBASE_MODE)

Set the default index tag mode.

XB_IX_DBASE_MODE - DBASE Compatibility mode
1) For index tags defined as unique, dBASE allows mutliple records in a table all having the same key, but only the first entry in the table with the unique key will be referenced in the unique index.
2) Deleted DBF records are not deleted from the associated tag file.
3) This is the system default.

XB_IX_XBASE_MODE - XBase mode
1) For index tags defined as unique, the library will throw an error when an attempt is made to add/update a record with a duplicate key.
2) Deleted DBF records are deleted from the associated index tag.

Note
When switching between modes, reindex any affected index tags.
