Detailed Description

template<class xbNodeType>
class xb::xbLinkListNode< xbNodeType >

xbLinkNode class.

This class provides a template for defining nodes that are used by the xbLinkList and xbLinkListOrd classes.

Nodes can be used for storing any data type, and then linked together with one of the linked list classes.
