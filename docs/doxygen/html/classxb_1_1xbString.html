Detailed Description

xbString is a basic string class with all the functionality one would expect in a string class.

For purposes of the xbString class, a string is defined as a variable sized array of one byte characters terminated with a null (0x00 or \0) byte.

This version of the xbString class does not support wide (wchar_t) characters at this time.

This string class handle strings in a 1-based (not 0 based) fashion. Any string routines taking an offset use a 1-based value. That is, the first position of the string is position 1, not 0. Position 1 (not 0) is considered the first position in a string. A return of 0 would indicate a not found condition. A return of 1, would be the first byte.
