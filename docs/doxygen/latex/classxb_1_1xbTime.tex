\doxysubsection{Detailed Description}
\mbox{\hyperlink{classxb_1_1xbTime}{xb\+Time}} class. 

The \mbox{\hyperlink{classxb_1_1xbTime}{xb\+Time}} class handles time values.

Times are stored internally in a long integer field with the following formula\+: Hours $\ast$ 3600000L + Minutes $\ast$ 60000L + Seconds $\ast$ 1000L;

Methods for setting the default time format at the system level\+: \begin{DoxyVerb}xbXBase::SetDefaultTimeFormat()
xbXBase::GetDefaultTimeFormat()
xbTime::SetDefaultTimeFormat()
xbTime::GetDefaultTimeFormat()
\end{DoxyVerb}
 Methods for setting the time format at the instance level (over ride system level default). \begin{DoxyVerb}xbTime::SetTimeFmt()
xbTime::GetTimeFmt()
\end{DoxyVerb}
 The system default time format is HH\+:\+MM\+:SS (\%H\+:\%M\+:\%S).

Time format specifiers (for applicable class methods).~\newline
~\newline
 \%H -\/ is replaced by the hour (24 hour clock) as a decimal number (00-\/23)~\newline
 \%I -\/ is replaced by the hour (12 hour clock) as a decimal number (01-\/12)~\newline
 \%M -\/ is replaced by the minute as a decimal number (00-\/59)~\newline
 \%S -\/ is replaced by the second as a decimla number (00-\/59)~\newline
 \%p -\/ is replaced by the locale\textquotesingle{}s equivalent of AM/\+PM with 12 hour clock~\newline


\begin{DoxyNote}{Note}
The \mbox{\hyperlink{classxb_1_1xbTime}{xb\+Time}} class is available when the XB\+\_\+\+DBF5\+\_\+\+SUPPORT option is compiled into the library. 
\end{DoxyNote}
