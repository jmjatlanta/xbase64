\doxysubsection{Detailed Description}
\mbox{\hyperlink{classxb_1_1xbBlockRead}{xb\+Block\+Read}} class. 

The \mbox{\hyperlink{classxb_1_1xbBlockRead}{xb\+Block\+Read}} class can be used to read blocks of records at one time rather than one record at a time for perfomance improvements when reading a DBF file sequentially.~\newline
 ~\newline
 See program xb\+\_\+dumprecs.\+cpp for an example on how to use block read processing.

Get record for specified record number. 

This method retrieves a record from read block buffer and copies it into the record buffer. If the record is not in the current block, the routine calls Get\+Block\+For\+Rec\+No to load the currect block from disk.
