\doxysubsection{Detailed Description}
\mbox{\hyperlink{classxb_1_1xbFilter}{xb\+Filter}} class. 

The x\+Filter class is used to establish a data filter on a given table (DBF file). 
\begin{DoxyEnumerate}
\item A filter can be established to limit selected records in a query. 
\item Similar to a {\itshape WHERE} clause in SQL. 
\item The filter logic uses the expression module for processing. 
\item Any of the expression routines that return a logical value can be used to establish a filter condition. 
\item See Chapter four or the xb\+\_\+ex\+\_\+expression sample program for more information on valid expressions. 
\end{DoxyEnumerate}
