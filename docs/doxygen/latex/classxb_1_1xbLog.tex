\doxysubsection{Detailed Description}
\mbox{\hyperlink{classxb_1_1xbLog}{xb\+Log}} class. 

The \mbox{\hyperlink{classxb_1_1xbLog}{xb\+Log}} class handles message logging functions.

If logging is enabled in an application, error messages are printed in the log file.

Once a logfile reaches a predefined size, the logfile is closed, renamed and opened.

The logging functions can be accessed through the \mbox{\hyperlink{classxb_1_1xbXBase}{xb\+XBase}} class. Each application has one of those.


\begin{DoxyItemize}
\item Error and message logging. 
\item Rolling log files. When log file fills, full file is closed and new file is opened. 
\item Configurable size. 
\item Configurable name. 
\item Configurable location. 
\end{DoxyItemize}
