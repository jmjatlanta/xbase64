\doxysubsection{Detailed Description}
\subsubsection*{template$<$class xb\+Node\+Type$>$\newline
class xb\+::xb\+Link\+List$<$ xb\+Node\+Type $>$}
\mbox{\hyperlink{classxb_1_1xbLinkList}{xb\+Link\+List}} class. 


\begin{DoxyItemize}
\item Template based class, supports a list of any object type 
\item Supports additions at the beginning or end of the list 
\item Support deletions at the beginning or end of the list 
\item Can be used to support a {\itshape stack data structure} by always adding to the end and removing from the end 
\item xb\+LLN is a linked list node of type xb\+Node\+Type 
\item Class is available if XB\+\_\+\+LINKLIST\+\_\+\+SUPPORT is compiled in library 
\end{DoxyItemize}
