\subsubsection*{template$<$class xb\+Node\+Type$>$\newline
class xb\+::xb\+Link\+List\+Node$<$ xb\+Node\+Type $>$}
xb\+Link\+Node class. 

This class provides a template for defining nodes that are used by the \mbox{\hyperlink{classxb_1_1xbLinkList}{xb\+Link\+List}} and \mbox{\hyperlink{classxb_1_1xbLinkListOrd}{xb\+Link\+List\+Ord}} classes.

Nodes can be used for storing any data type, and then linked together with one of the linked list classes.
